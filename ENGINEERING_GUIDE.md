# Core Engineering Standards & Guidelines

## Purpose
Establishes the foundational rules, quality baselines, and peer-review requirements expected from engineers contributing to the NEXUS MESH protocol.

## Scope
Covers local workflows, branching strategies, and lint rules.

## Basic Pipeline
All pull requests must compile successfully without warnings and pass automated unit tests in the CI build before merging.

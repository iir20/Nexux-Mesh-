# Android Keystore Cryptographic Hardware Key Management

## Purpose
Details the physical hardware key isolation model, AES key creation, and biometric prompt bindings.

## Scope
Details Keystore alias entries, cipher wrapping, and cryptographic isolation.

## Security Level
Generates secure keys inside the hardware-backed Trusted Execution Environment (TEE) or StrongBox key store where physical chip architecture isolates private key extraction.

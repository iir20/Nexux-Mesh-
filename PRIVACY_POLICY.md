# Offline-First Privacy Policy & Consent

## Purpose
Formulates legal protections, transparency standards, and privacy disclosures regarding user telemetry and local data handling.

## Scope
Covers local databases, telemetry loops, network packet transmission, and analytical statistics.

## Offline-First Guarantee
NEXUS MESH is built on an offline-first architecture:
- **No Remote Harvesting**: All message transactions, logs, contacts, and feed metadata are stored strictly on-device.
- **No IP Tracking**: The system randomizes BLE advertising MAC addresses to block device tracking.
- **Encrypted Local Storage**: Local state databases are compiled to prevent raw SQLite file indexing.

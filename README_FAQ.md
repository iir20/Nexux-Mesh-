# Frequently Asked Questions (FAQ) & Troubleshooting

## Purpose
Resolves common operation problems, compatibility issues across hardware, and battery consumption profiles.

## Scope
Details battery optimizer bypasses, Bluetooth GATT error mitigations (such as GATT Error 133), and database growth.

## Questions Covered
1. **Why does BLE connection fail with Error 133 on Android 12?**
   - This occurs due to stale system connection caches. The application automatically recycles the BLE state to resolve this.
2. **Does this application run without root?**
   - Yes, NEXUS MESH uses standard Android public SDK APIs and requires no system rooting.

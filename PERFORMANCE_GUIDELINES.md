# Performance Targets & Optimization Standards

## Purpose
Defines memory allocations, garbage collection avoidance guidelines, and layout optimizations for Jetpack Compose.

## Scope
Target:
- Keep frame rates above **60 FPS** during list scroll actions.
- Limit database read execution times under **20ms**.

# Cryptographic Encryption & Signature Spec

## Purpose
Specifies exact library versions, curve configurations, AES key wrapping, and integrity digests.

## Algorithms
- Asymmetric Curve: NIST P-256 (`secp256r1`)
- Symmetric Bulk Cipher: AES-256-GCM
- Hash Standard: SHA-256
